Privacy Policy

Last updated: January 2025

1. Data We Collect

  • Server ID and configured channel ID
  • User ID (for wishlist and DM preferences)
  • Wishlist game titles
  • Language and platform preferences

2. How We Use Data

  • To send deal notifications to configured channels
  • To send personal DM alerts for wishlist matches
  • To remember your preferences

3. Data Storage

  • Data is stored securely in a PostgreSQL database
  • We do not sell or share your data with third parties
  • Payment processing is handled by Stripe (see Stripe's privacy policy)

4. Data Retention

  • Server data is kept while the bot is in your server
  • User data can be deleted upon request
  • Removing the bot deletes server configuration

5. Your Rights

  • Request a copy of your data
  • Request deletion of your data
  • Opt out of DM notifications anytime via /settings

6. Children's Privacy

LootGoblin is not intended for users under 13 years of age.

7. Contact

For privacy concerns, contact us via Discord.
